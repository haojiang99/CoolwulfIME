## English

### Overview

**Coolwulf IME** is an advanced physical keyboard input method for Android devices, designed specifically for users who prefer typing on hardware keyboards. It features intelligent Chinese Pinyin input with CC-CEDICT dictionary support (124,117+ entries), user learning capabilities, customizable layouts, and powerful modifier key support.

Originally developed as Pastiera, this keyboard has evolved to provide exceptional support for both English and Chinese input on physical keyboards.

### Key Features

- **Physical Keyboard Optimized**: Built from the ground up for hardware keyboards
- **Intelligent Pinyin Input**: Full Chinese Pinyin support with 124,117+ dictionary entries from CC-CEDICT
- **User Learning**: Automatically learns your typing patterns and prioritizes frequently used characters
- **Multi-language Support**: English, Chinese, Italian, French, German, Polish, Spanish
- **Customizable Layouts**: Support for multiple keyboard layouts including Cyrillic and custom mappings
- **Auto-correction**: Built-in auto-correction for multiple languages
- **Modifier Keys**: Full support for Alt, Ctrl, Shift with lock modes
- **Navigation Mode**: Quick navigation using letter keys as shortcuts (double-tap Ctrl)
- **LED Indicators**: Visual status bar indicators for modifier keys
- **Voice Input**: Integrated speech-to-text support (Alt+Ctrl shortcut)
- **Launcher Shortcuts**: Assign keys to launch apps from home screen

### Installation

1. **Download and Install**: Install the Coolwulf IME APK on your Android device
2. **Enable the Keyboard**:
   - Open Coolwulf IME app
   - Tap "Enable Coolwulf IME" button
   - In Android Settings, enable "Coolwulf IME Physical Keyboard"
3. **Select as Default**:
   - Tap "Choose input method" in the app
   - Select "Coolwulf IME Physical Keyboard" as your default keyboard
4. **Connect Your Keyboard**: Connect a physical keyboard via Bluetooth or USB

### Using English Input

#### Basic Typing

1. Tap any text field to activate the keyboard
2. Type normally on your physical keyboard
3. Text appears in the input field

#### Smart Features

- **Auto-Capitalize**: Automatically capitalizes the first letter in new sentences
- **Double Space to Period**: Tap space twice to insert a period and capitalize the next letter
- **Auto-Correction**: Typos are automatically corrected based on selected language dictionaries

#### Modifier Keys

The keyboard features advanced modifier key support:

- **Shift**:
  - Tap once: Next letter is capitalized (one-shot mode)
  - Double-tap: CAPS LOCK (red LED indicator)

- **Alt**:
  - Tap once: Access special characters for next key (one-shot mode)

  - Tap once: Use for shortcuts like Ctrl+C, Ctrl+V (one-shot mode)

### Advanced Features

#### Navigation Mode

Double-tap **Ctrl** when not in a text field to activate Navigation Mode:
- Letter keys become navigation shortcuts
- **ESDF** or **IJKL**: Cursor movement

#### SYM Keyboard

Press the **SYM** key to access emoji and special characters:
- **First press**: Emoji page (each letter mapped to an emoji)
